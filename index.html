Project on Console

Visual Studio Code is a coding platform that can be downloaded in 5 minutes and used by anyone! You can use Visual Studios to code a website that can be used by millions of people, like Wikipedia.

The console screen is a coder's best friend. It can be used to figure out any issues with the code and it will specifically tell what the issue is and in what line of code it is in. The console screen can be opened by right-clicking on the web page and clicking inspect. Then, you can choose console and see if there are any issues with the code.


Right click anywhere on the screen and click inspect, then click on the console tab so that you can see the console screen.
